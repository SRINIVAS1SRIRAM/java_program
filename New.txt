To import files to the github
=============================

1.git config --global user.name "SRINIVAS1SRIRAM"
2.git config --global user.email srinivas1sriram@gmail.com
3.git clone (copy the repository link and paste it)
4.now go to the location of the folder and type the command git init
5.git add (filename)	//to add the file
6.git status 	//to view the status
7.git commit -m "comments" (filename) //to commit the file
8.git push -u origin master	//to push the file


if the file is edited repeat step 7 and 8 .
